freestream-0.3.0.0: Effectful, iteratee-inspired stream processing based on a free monad.

This package provides the Task monad transformer, allowing any monad computation to become a stream processing pipeline. A computation of type Task a b m r consumes values of type a and produces values of type b.

Examples and more information may be found at https://github.com/gatlin/FreeStream.
